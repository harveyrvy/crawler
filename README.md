## Crawler

To set up this project:

## Clone the repo

```
$ git clone https://github.com/harveyrvy/crawler.git
```

## Start Backend

```
$ cd backend
$ go run .

```

## Start Frontend

```
$ cd .. && cd frontend
$ npm install
$ npm run build

```

Then navigate to [http://localhost:5173/](http://localhost:5173/).
